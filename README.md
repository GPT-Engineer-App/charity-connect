# charity-connect

A website that can help charities and local organisations find volunteers. Allow charities to post messages detailing the volunteering opportunities available. Also allow volunteers to post that they are available to volunteer

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/charity-connect.git
cd charity-connect
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
